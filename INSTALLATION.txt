1) Make sure you have node version 22.x installed

    node --version


2) Clone and run the installation which will ask to install to a subdirectory called "tradingview-alerts-home"

    git clone https://github.com/best-trading-indicator-tools/tradingview-alerts.git
    cd tradingview-alerts/create-tradingview-alerts-home
    npm install && npm run build
    node dist/index.js
